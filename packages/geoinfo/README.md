## keplerjs:geoinfo

keplerjs plugin geoinfo


## API requirements

Geonames API user:
http://www.geonames.org/export/web-services.html
https://www.geonames.org/export/credits.html

Ipinfodb API key:
http://ipinfodb.com/ip_location_api.php

*Settings.json*
```
"geoinfo": {
    "geonamesUser": "<USER>",
    "ipinfodbKey": "<KEY>"
}
```
